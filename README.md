# Online-test
Based on Node.js Online examination system </br>
</br>
这是一个基于Node.js语言开发的学校选课B/S结构系统</br>
系统分为的用户分为：学生与教师</br>
 （1） 学生功能：</br>
      &nbsp;&nbsp; 1.注册登录账号</br>
      &nbsp;&nbsp;  2.对自己的信息加以查询</br>
      &nbsp;&nbsp; 3.进行选课</br>
      &nbsp;&nbsp; 4.参加选课内容测试</br>
 </br>
 （2）教师功能：</br>
       &nbsp;&nbsp;1.注册登录账号</br>
       &nbsp;&nbsp; 2.对学生信息加以查询</br>
       &nbsp;&nbsp; 3.上传试题</br>
